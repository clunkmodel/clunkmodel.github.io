Drawing Recognition

In Clunk's reflection tool, artificial intelligence (AI) recognizes drawings from students to select icons that can be placed on the scene to guide a student in his thinking. As an example, a Convolutional Neural Network for this AI was trained to recognize 100 classes using the quick draw dataset. The model has a test accuracy of 90.34%. The example drawing canvas shows how Clunk's drawing recognition might work in the context of the ocean example scene as deployed in the demonstrator.

The model was based on the example given by Zaid Alyafeai.
